Memahami Kubernetes

calendar_today Feb 2024
timer 15 min read

1. Pengenalan Kubernetes

Kubernetes (juga dikenal sebagai K8s) adalah sistem open-source untuk mengotomatisasi deployment, scaling, dan manajemen aplikasi yang dikontainerisasi. Awalnya didesain oleh Google, Kubernetes kini dikelola oleh Cloud Native Computing Foundation (CNCF).

2. Konsep Inti

Untuk memahami K8s, Anda perlu familiar dengan beberapa komponen dasar:

  • Pod: Unit terkecil yang dapat di-deploy. Biasanya berisi satu atau lebih container.
  • Node: Mesin fisik atau virtual tempat Pod berjalan.
  • Cluster: Kumpulan Node yang dikelola oleh control plane.
  • Deployment: Mengelola replikasi dan update dari Pod.

3. Contoh Deployment

Mari kita lihat contoh file YAML dasar untuk mendeploy aplikasi Nginx sederhana.

apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx-deployment
  labels:
    app: nginx
spec:
  replicas: 3
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx
        image: nginx:1.14.2
        ports:
        - containerPort: 80

Gunakan perintah kubectl apply -f deployment.yaml untuk mengaplikasikan konfigurasi di atas.
