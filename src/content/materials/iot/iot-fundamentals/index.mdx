# IoT Fundamentals

Pelajari dasar-dasar Internet of Things! Bangun aplikasi IoT hybrid yang tidak sepenuhnya bergantung pada internet.

## Apa yang Akan Kamu Pelajari

- **Event Pattern**: Pola event-driven untuk IoT
- **In-Memory Database**: Redis untuk penyimpanan cepat
- **Message Broker**: RabbitMQ untuk komunikasi antar komponen
- **Logging**: Pencatatan log untuk debugging
- **Automation Testing**: Pengujian aplikasi IoT
- **Project Vending Machine**: Proyek praktis IoT

## Prerequisites

- Dasar pemrograman
- Raspberry Pi atau simulator
- Koneksi internet untuk setup awal
