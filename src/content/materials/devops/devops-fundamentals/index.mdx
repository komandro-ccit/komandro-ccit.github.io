# DevOps Fundamentals

Pelajari dasar-dasar DevOps dari nol! Mulai dari Linux, virtualization, Docker, hingga cloud computing.

## Apa yang Akan Kamu Pelajari

- **Introduction to DevOps**: Pengertian dan manfaat DevOps
- **Linux & Virtualization**: Dasar sistem operasi Linux
- **Build & Deployment**: Proses build dan deploy aplikasi
- **Docker**: Containerization untuk aplikasi modern
- **Docker Compose**: Orchestration container
- **Cloud Computing**: Pengenalan layanan cloud
- **LocalStack**: Testing layanan cloud secara lokal
- **Infrastructure as Code**: Otomasi infrastruktur

## Prerequisites

- Laptop dengan minimum 8GB RAM
- Koneksi internet stabil
- Terminal/command line basics
