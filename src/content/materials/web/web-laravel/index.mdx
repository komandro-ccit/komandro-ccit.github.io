# Laravel Framework

Pelajari Laravel, framework PHP paling populer untuk membangun aplikasi web modern yang scalable.

## Apa yang Akan Kamu Pelajari

- **Laravel Basics**: Struktur folder dan konsep dasar
- **Routing**: Mengelola URL dan endpoint
- **Eloquent ORM**: Interaksi database yang elegant
- **CRUD Operations**: Implementasi operasi database lengkap
- **Multi-Auth**: Sistem autentikasi multi-role dengan Breeze

## Prerequisites

- Pemahaman PHP dasar dan MVC
- Web Development Advance atau pengetahuan setara
- Composer terinstall
