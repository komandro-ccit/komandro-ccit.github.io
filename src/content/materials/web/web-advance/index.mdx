# Web Development Advance

Masuk ke dunia backend development dengan PHP native! Pelajari arsitektur MVC, templating, dan operasi database.

## Apa yang Akan Kamu Pelajari

- **PHP Fundamentals**: Dasar-dasar pemrograman PHP
- **MVC Architecture**: Pola desain Model-View-Controller
- **Templating**: Memisahkan logika dan tampilan
- **Database CRUD**: Operasi Create, Read, Update, Delete dengan MySQL
- **Authentication**: Sistem login dan session management

## Prerequisites

- Pemahaman HTML, CSS, dan JavaScript
- Web Development Intermediate atau pengetahuan setara
