# Web Development Intermediate

Tingkatkan skill web development-mu ke level berikutnya! Pelajari Bootstrap untuk layout cepat dan JavaScript untuk interaktivitas.

## Apa yang Akan Kamu Pelajari

- **Bootstrap**: Framework CSS untuk layout responsif yang cepat
- **JavaScript**: Bahasa pemrograman untuk web interaktif
- **DOM Manipulation**: Mengontrol elemen HTML dengan JavaScript

## Prerequisites

- Pemahaman dasar HTML dan CSS
- Web Development Beginner atau pengetahuan setara
