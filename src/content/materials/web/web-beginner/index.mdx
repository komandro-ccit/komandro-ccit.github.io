# Web Development Beginner

Mulai perjalananmu di dunia web development! Course ini akan membawamu memahami fondasi pembuatan website modern.

## Apa yang Akan Kamu Pelajari

- **HTML**: Struktur dan elemen dasar halaman web
- **CSS**: Styling dan layout untuk tampilan yang menarik
- **Semantic HTML**: Menggunakan tag yang tepat untuk konteks yang tepat
- **Bootstrap**: Framework CSS populer untuk layout responsif

## Prerequisites

- Laptop dengan koneksi internet
- Browser modern (Chrome/Firefox)
- Text editor (VS Code direkomendasikan)
