# Dart Fundamentals

Pelajari bahasa pemrograman Dart dari nol! Dart adalah fondasi untuk pengembangan aplikasi Flutter.

## Apa yang Akan Kamu Pelajari

- **Dart Introduction**: Pengenalan bahasa Dart dan setup environment
- **Basic Dart**: Variabel, tipe data, operator, dan control flow
- **Dart OOP**: Object-Oriented Programming di Dart
- **Advance Dart**: Generic, asynchronous programming, dan koleksi

## Prerequisites

- Laptop dengan minimum 8GB RAM
- VS Code atau Android Studio
- Pengetahuan programming dasar (opsional)
