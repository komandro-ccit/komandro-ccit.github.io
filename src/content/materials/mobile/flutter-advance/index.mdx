# Flutter Advanced

Tingkatkan skill Flutter-mu ke level profesional! Pelajari state management lanjutan, networking, database, autentikasi, animasi, dan testing.

## Apa yang Akan Kamu Pelajari

- **State Management Lanjutan**: Provider dan Riverpod
- **Networking**: HTTP requests dan API integration
- **Database**: Penyimpanan data lokal dan cloud
- **Authentication**: Sistem autentikasi dengan Supabase
- **Animation**: Membuat animasi yang menarik
- **Automation Testing**: Pengujian otomatis

## Prerequisites

- Flutter Beginner atau pengetahuan setara
- Dasar Dart programming
- Flutter SDK terinstall
