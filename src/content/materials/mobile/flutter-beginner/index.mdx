# Flutter Beginner

Mulai perjalanan mobile development-mu dengan Flutter! Bangun aplikasi cross-platform yang cantik dan fungsional.

## Apa yang Akan Kamu Pelajari

- **Widget Introduction**: Memahami sistem widget Flutter
- **Layout Introduction**: Membangun layout responsif
- **Navigation & Routing**: Berpindah antar layar
- **State Management**: Mengelola data dalam aplikasi
- **Interaction & Input**: Menangani input pengguna

## Prerequisites

- Dasar Dart programming
- Flutter SDK terinstall
- VS Code atau Android Studio dengan Flutter extension
