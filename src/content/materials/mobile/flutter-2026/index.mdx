# Flutter Fundamentals 2026

Selamat datang di kurikulum lengkap Flutter! Course ini dirancang untuk membawa kamu dari pemula hingga bisa membangun aplikasi Flutter yang production-ready.

## Apa yang Akan Kamu Pelajari

- **Dart Programming**: Bahasa pemrograman modern untuk Flutter
- **Widget Fundamentals**: Membangun UI dengan widget
- **State Management**: Mengelola data dalam aplikasi
- **Navigation**: Berpindah antar layar
- **Networking**: Mengambil data dari API
- **Local Storage**: Menyimpan data di perangkat

## Struktur Course

Course ini terdiri dari 5 modul utama:

1. **Setup & Installation** - Persiapan environment development
2. **Hello World** - Aplikasi Flutter pertama
3. **Widget Basics** - Stateless vs Stateful widgets
4. **State Management** - Provider dan Riverpod
5. **API Integration** - Working with REST APIs

## Prerequisites

- Basic programming knowledge
- Laptop with minimum 8GB RAM
- VS Code atau Android Studio

Let's build amazing apps! 🚀
